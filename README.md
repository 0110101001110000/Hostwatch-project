

# Hostwatch-project


Pipeline de dados para análise de hospedagens com PostgreSQL e dashboards em Python. 

**Tecnologias:**

*    **Linguagem:** Python
*    **Banco de Dados:** PostgreSQL
*    **Bibliotecas:** Selenium, Pandas, Matplotlib, ...
